# Raspberry Pi Pico W

## Flash

Flash partitions

```
|------------------------------------------------|
|          A         | B |     C     |     D     |
|--------------------|---|-----------|-----------|
|       1008K        |16K|   512K    |   512K    |
|------------------------------------------------|
                     |--------- flash.c ---------|

|--------- 1MB ----------|---------- 1MB---------|
|---------------------- 2MB ---------------------|
```

- A : Binary (firmware)
- B : Storage (key-value database)
- C : User program (js)
- D : File system (lfs)
  (Total : 2MB)
